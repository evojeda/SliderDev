🎞️ SliderDev

Projeto de slider interativo de imagens desenvolvido com HTML, CSS e JavaScript.
Ao clicar nos botões de navegação, o usuário pode alternar entre diferentes imagens, criando um efeito de apresentação visual simples e elegante.
O objetivo do projeto foi praticar manipulação do DOM, eventos de clique e transições visuais, conceitos fundamentais do desenvolvimento front-end.

🔗 Acesse o projeto:
https://evojeda.github.io/SliderDev/

🚀 Tecnologias utilizadas

- HTML5 – estrutura da página
- CSS3 – layout, estilização e animações
- JavaScript – lógica do slider e interação do usuário

🎨 Funcionalidades

- ✔️ Slider de imagens interativo
- ✔️ Botões de navegação para trocar as imagens
- ✔️ Layout responsivo
- ✔️ Transição visual entre as imagens
- ✔️ Estrutura simples e organizada para aprendizado
